## Local Development

First, `install` dependencies, then you need to `build` or start `watch mode` to compile code.

Next step you can:

- Start Dev Server
- Start Storybook server

```
// Install
# yarn

// Build
yarn build

```

### Watch mode

```
# yarn watch:styling
# yarn watch:ts
```

## Storybook server

```
# yarn storybook
```
